2.使用教程 

    1.安装油猴 
    2.将脚本拷贝到油猴脚本中
    3.修改课程你要选择的课程名称
    
    Ps:如果你没有油猴
        1.修改脚本中你要选的课程
        2.ctrl+A => ctrl+c 复制代码
        3.按F12 => 打开控制台 => 粘贴脚本 => 按enter即可
